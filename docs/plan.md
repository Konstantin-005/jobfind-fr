- **Шаги выполнения**:

  - [ ] Добавить клиентскую фильтрацию по отраслям и названию через query-параметры и lazy load («Показать ещё»)
  - [ ] Реализовать SSR-страницу `/companies/{id}` с вкладками «Вакансии» и «Инфо»
