## [2025-12-31] - Предложение вакансии по резюме
### Добавлено
- Кнопка «Предложить вакансию» на публичной странице резюме `/resume/[link_uuid]` для работодателя
- Модалка выбора вакансии работодателя (список по `/api/companies/jobs`) с подтверждением отправки предложения по выбранной вакансии и `resume_link_uuid`
- Возможность отправить предложение из списка резюме на странице `/resume` через карточку резюме

### Изменено
- При успешной отправке предложения по вакансии отображается уведомление «Предложение по вакансии отправлено. Соискателю пришло письмо и сообщение в чат.» и, при наличии `chat_room_id` в ответе, кнопка «Перейти в чат» с переходом в `/chat/[chat_room_id]`
- В тело запроса `POST /api/jobs/{job_id}/offer` добавлено поле `message_text` (до 5000 символов), содержащее индивидуальное сообщение работодателя для первого сообщения в чат и письма соискателю
- Для повторных попыток отправки предложения по той же паре «вакансия + резюме» при ответе 400 с сообщением о дубликате кнопка «Предложить» блокируется (disabled)

### Исправлено
- Отображение и поведение кнопки «Предложить вакансию» завязаны на роль `employer`; для других ролей она не показывается, а ошибки 401/403 корректно обрабатываются на фронте (уведомление и редирект на логин)
- Ошибка 400 с текстом «Длина сообщения не должна превышать 5000 символов» подсвечивается под полем ввода текста предложения, без общего системного сообщения
